# no-trailing-spaces

:dress: The `extends: 'stylistic'` property in a configuration file enables this rule.

Disallow trailing whitespace at the end of lines.

## Examples

This rule **forbids** the following:

```hbs
<div>test</div>//••
//•••••
```

This rule **allows** the following:

```hbs
<div>test</div>//
//
```
